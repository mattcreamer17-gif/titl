# Titl landing page

A bespoke, prospect-facing landing page for Titl (titl.co), built in the company's own cyan / DM Sans / pale-gradient register so it reads like their own marketing team produced it. The page carries the 60/30/10 read on their pipeline, cites the Feeld scope reference and ICM methodology, names the NLP Logix partnership with the required credibility line, and drops the prospect into a 30-minute call with Matt Creamer (CRO) via the Calendly link. Logo, colors, and typography pull directly from the Titl brand pack; Eduba appears only in the footer.
